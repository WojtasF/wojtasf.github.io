Website for local coffe shop.

Ok I will start by saying that I’m not good designer, so I recently bought package of templates (BeTheme by muffingroup) and I came up with an idea of making websites from scratch based on their templates, using different frameworks, librariers etc. So here comes my first nice looking website!

What I’ve learned:

  • More SASS (I think I won’t come back to normal css, it’s great!)
  • More JS
  • More practise with Bootstrap 4

What I’ve used to make this website:

  • Bootstrap - CSS Framework;
  • SASS - CSS pre-compiler;
  • JQuery - JS Framework;
  • Animate - CSS Library for animations
  • Lightbox - library for Image Gallery
